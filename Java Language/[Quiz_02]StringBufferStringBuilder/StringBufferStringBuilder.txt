Problem Description

輸入一個字串，並將字串其中的片段取代，再將其反轉輸出。

Note: 取代後的字串將不包含原始字串。
Input Format

先輸入一個數字 N，代表總共有多少組測試資料。每組測資都會有三行字串。先輸入一行原

始字串，再輸入需要被取代的字串，最後輸入取代用的字串，所有字串均允許有空格輸入。
Output Format

對每一組測試資料，請輸出最後取代完成字串的反轉字串。

Note: 取代後的字串將不包含原始字串。
注意！本題須使用StringBuffer或StringBuilder及

使用其methods做字串的反轉與取代。
