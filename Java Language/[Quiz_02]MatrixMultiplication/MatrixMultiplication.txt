Problem Description

寫出一個矩陣相乘的程式，計算最後矩陣答案。
Input Format

先輸入數字 N，代表總共有多少組測試資料，每組測資都會依序輸入矩陣的大小，再輸入矩

陣的數值。假設輸入為

A =

1 2 3

−1 0 4

B =

1 2 3 3

0 −4 4 0

−1 0 −2 1

會先輸入第一個矩陣大小 2 3，再分別輸入每列 (1 2 3)，(-1 0 4 ) (請參考 Sample Input) 再

輸入第二個矩陣大小 3 4，分別輸入每列 (1 2 3 -3)，(0 -1 4 0)，(-1 0 -2 1)，所有的輸入皆為

Integer，所有輸入的矩陣測資都可以相乘。
Output Format

輸出最後的答案，每組測試資料間以換行符號分隔，但最後一組測資則沒有多一個換行符號

(請參考 Sample Output)
